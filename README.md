# Roblox.Tools

Assortment of Roblox client modification tools, updated from time to time

```
Roblox.Tools
↳ Roblox.DiogenesConverter
↳ Roblox.ScriptSigner
↳ Roblox.KeyGenerator
```

## Build

1. Open `Roblox.Tools.sln`
2. Run a NuGet package restore
3. Build Solution
